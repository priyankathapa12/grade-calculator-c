# Student Grade Calculator (C Program)

This repository contains a simple and beginner-friendly **C program** that calculates:
- Total marks
- Average marks
- Final grade (A, B, C, D, F)
It is a useful project for students learning C programming, conditional statements, and input/output handling.
#Features
- Takes input for 3 subjects  
- Calculates total and average  
- Displays final grade based on average  
- Easy to understand and modify  
#How It Works
The program uses:
- `scanf()` for input  
- Basic arithmetic operations  
- `if...else` conditions for grading
